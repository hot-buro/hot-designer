# Hot Designer

Розроблено **HotBuro**.

Один плагін — усі ключові дизайнерські скіли. Встановлення: відкрити `.plugin` файл у Cowork і натиснути «Встановити».

## Що всередині (29 скілів)

| Скіл | Джерело | Призначення |
|---|---|---|

| ui-ux-pro-max, design, design-system, ui-styling, brand, banner-design, slides | [nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) | База знань: 50+ стилів, 161 палітра, 57 пар шрифтів, банери, слайди, брендинг |

## Примітки

- Hooks з Impeccable і Superpowers (live-mode, session-start) не включені — вони специфічні для Claude Code. Скіли працюють повністю.
- Кожен скіл зберігає LICENSE оригінального репозиторію (MIT). Усі права належать авторам.
- Оновлення: перезбирається з тих самих репозиторіїв.
